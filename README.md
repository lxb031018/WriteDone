# WriteDone 🖊️

一个极简、省电的 Android 待办 + 番茄计时器应用。

## 功能

- **📌 便利贴** — 输入待办事项，计时结束后自动归档至「已完成」；归档后仍可追加备注
- **⏱️ 计时器** — 支持普通计时 / 番茄计时（25 分钟累积后提示休息）；平放手机且便利贴有内容时自动开始计时
- **📅 日历** — 右滑进入，长按日期复制当日摘要，可粘贴至 AI 助手分析生活质量
- **🎨 主题** — 支持系统 / 浅色 / 深色模式
- **🔋 省电** — 常规使用几乎不耗电；横屏摆件模式功耗略增
- **🔔 权限** — 建议授予通知、精确闹钟、自启动、锁屏通知等权限以保障提醒功能

## 隐私承诺

WriteDone **仅读取本地数据**，不会上传至云端，无需网络权限。

## 技术栈

- Kotlin + Jetpack Compose
- Hilt (DI)
- Room (本地存储)
- Coroutines + Flow
